## 💡 QUICK DEPLOYMENT GUIDE

### If using Vercel (Recommended):

```bash
# 1. Install Vercel CLI
npm i -g vercel

# 2. Navigate to frontend directory
cd frontend

# 3. Deploy
vercel --prod

# 4. Vercel will:
#    - Build your Next.js app
#    - Enable HTTPS automatically
#    - Give you a production URL (trulyinvoice.vercel.app)
#    - You can later add custom domain (trulyinvoice.xyz)
```

### After Deployment:

1. **Get Live URL**: https://trulyinvoice.vercel.app (or custom domain)

2. **Google Search Console**:
   - Go to: https://search.google.com/search-console
   - Add property: trulyinvoice.xyz
   - Verify using HTML tag method
   - Copy verification code: `<meta name="google-site-verification" content="YOUR_CODE" />`
   - Update in `frontend/src/app/layout.tsx` (replace 'your-google-verification-code')
   - Redeploy

3. **Google Analytics**:
   - Go to: https://analytics.google.com
   - Create property: TrulyInvoice
   - Get Measurement ID: G-XXXXXXXXXX
   - Update in `frontend/src/lib/analytics.ts` (replace 'G-XXXXXXXXXX')
   - Redeploy

4. **Test Everything**:
   ```bash
   # Run Lighthouse
   https://pagespeed.web.dev/
   # Enter your URL
   
   # Validate Schema
   https://validator.schema.org/
   # Enter your URL
   
   # Test Mobile Friendly
   https://search.google.com/test/mobile-friendly
   # Enter your URL
   
   # Test Open Graph
   https://developers.facebook.com/tools/debug/
   # Enter your URL
   ```

5. **Submit Sitemap**:
   - In Google Search Console
   - Go to Sitemaps section
   - Submit: https://trulyinvoice.xyz/sitemap.xml
