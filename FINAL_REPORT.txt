PROBLEM:
After uploading invoices, dummy/test data was appearing instead of real invoices.

ROOT CAUSE:
1. Database contained 1 dummy invoice with user_id = NULL
2. Frontend query fetched user invoices + NULL user_id invoices
3. Result: Dummy invoice appeared for ALL users
